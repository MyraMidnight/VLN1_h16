#  'Skrá ný gögn'
```
#####################################################################
Bókhaldskerfið > Skrá ný gögn

        Veldu hvaða gögn þú ætlar að skrá:                              
            1. Starfsmaður  
            2. Flugvél      
            3. Vinnuferð    
            4. Áfangastaður 

[p] fara í 'birta gögn'                     [u] fara í 'uppfæra gögn'
```
## Skrá nýjan starfsmann
```
#####################################################################
Bókhaldskerfið > Skrá ný gögn > Nýr starfsmaður

        Sláðu inn eftirfarandi upplýsingar:
            1. Fullt nafn (*)
            2. Kennitala (*)
            3. Heimilisfang
            4. Heimasími
            5. Farsími

* Nauðsynlegar upplýsingar sem verður ekki breytt seinna

[b] hætta við       [v] vista
```
Input line
```
Sláðu inn fullt nafn:
Sláðu inn kennitölu:
Sláðu inn heimilisfang:
Sláðu inn heimasíma:
Sláðu inn farsíma:
```
Birtir síðan innslegnar upplýsingar og spyr svo hvort eigi að vista.
```
#####################################################################
Bókhaldskerfið > Skrá ný gögn > Nýr starfsmaður

    Innslegnar upplýsingar
        1. Fullt nafn: Jón Jónson
        2. Kennitala: *******-****
        3. Heimilisfang: Staðargata 3
        4. Heimasími: *******
        5. Farsími: *******

Eru þessar upplýsingar réttar (y/n)?
```
```
Nýr starfsmaður hefur verið skráður í kerfið.
```
