
# 'Birta gögn'
```
#####################################################################
Bókhaldskerfið > Birta gögn

        Veltu hvaða gögn þú vilt birta:
            1. Starfsfólk
            2. Vinnuyfirlit (vaktir)
            3. Flugvélar 
            4. Vinnuferðir
            5. Áfangastaðir

[n] fara í 'skrá gögn'                      [u] fara í 'uppfæra gögn'
```
